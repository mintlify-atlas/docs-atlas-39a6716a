This integration flow runs on SAP Cloud Platform Integration (CPI) and synchronizes commission compensation data for active employees. It reads job records from SAP SuccessFactors, fetches commission totals from an SAP S/4HANA OData service, and writes the results back to a custom SuccessFactors object — all on an automated schedule.

## How it works

<Steps>
  <Step title="Timer fires on schedule">
    A cron-based timer starts the flow every 5 minutes, querying SuccessFactors for all employees with active job status (`emplStatus = 1465`).
  </Step>
  <Step title="Employee records are split">
    Each `EmpJob` record from SuccessFactors is split individually using an XPath General Splitter, so each employee is processed independently.
  </Step>
  <Step title="Commission data is fetched">
    For each employee, the flow calls the SAP S/4HANA OData service (`ZHR_CARTAS_SRV/CartaSet`) to retrieve total and average commission values.
  </Step>
  <Step title="Data is written back to SuccessFactors">
    The flow maps the commission data into the custom `cust_Info_Certificados` object and upserts it into SuccessFactors via OData V2.
  </Step>
</Steps>

  <Card title="Scheduled execution" icon="clock">

    Reads from and writes to both SAP SuccessFactors and SAP S/4HANA using OData V2.
